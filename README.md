# Florencia Reports

Internal reports dashboard for m5x2 property operations. GitHub Pages site with two report types:

- **Weekly Update** — team performance, operational workflow, budget vs. spent MTD, by location (Tacoma, Spokane, Tri-Cities)
- **P&L Assignments** — R&M-Repairs and R&M-Grounds budget vs. actual by property, plus appliance replacement breakdown

Live at: https://florencia83.github.io/Reports/
